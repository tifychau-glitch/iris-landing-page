FAQ

Common questions.

Do I need coding experience? +
No. Iris runs inside Claude Code, but you don't need to write code. The setup takes about 15 minutes and Iris walks you through every step conversationally.
What do I need to run Iris Pro? +
A Claude Code subscription (from Anthropic), and optionally a Pinecone account (free tier works) for persistent memory and a Telegram account for mobile access.
What's the difference between Core and Pro? +
Core gives you the accountability engine through Telegram — it's a hosted bot you can message right away, no setup required. Pro gives you the full AI operating system: 11 prebuilt skills, persistent memory, content creation, research, lead gen, task management, and the ability to build your own custom skills.
Is this a subscription? +
Core is free forever. Pro and Plus are one-time purchases with lifetime access. Enterprise is the only tier with a monthly component ($500/mo for ongoing maintenance and consultation).
What if I get stuck during setup? +
Iris literally walks you through setup step by step. But if you want hands-on help, the Plus tier includes a 90-minute 1:1 onboarding call and 30 days of private Slack support.
